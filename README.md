SAS Data Analysis Project

## Overview
This project involves analysing healthcare datasets using SAS, focusing on data preparation, transformation, and statistical analysis. The work is divided into two main tasks: patient data analysis and lung cancer mortality analysis.

---

## Task 1: Patient Data Analysis
This task focuses on analysing hospital patient data.

### What I did
- Imported raw data using INFILE and INPUT
- Structured and cleaned the dataset
- Created a variable for duration of hospital stay
- Used PROC FORMAT to label categorical variables
- Generated descriptive statistics using PROC MEANS
- Performed frequency analysis using PROC FREQ
- Created visualisations such as histograms and boxplots

### Outcome
- Analysed patient stay duration
- Explored relationships between admission year, funding type, and health conditions

---

## Task 2: Lung Cancer & Population Analysis
This task focuses on analysing lung cancer mortality data with population data.

### What I did
- Worked with two datasets: lung cancer deaths and population data
- Merged datasets using common variables (e.g. county)
- Calculated total deaths and total population
- Computed mortality rate (percentage) for each region
- Created summary tables for analysis

### Outcome
- Analysed lung cancer mortality rates across regions
- Compared population and death data to identify trends
