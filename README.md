# SAS_Patient_analysis
SAS project analysing patient data using statistical methods and visualisation
This project focuses on analysing hospital patient data using SAS. The aim was to prepare, transform, and analyse structured data to identify patterns and relationships.

## Tasks Performed
- Imported raw data using INFILE and INPUT statements
- Cleaned and structured the dataset
- Created new variables such as duration of hospital stay
- Applied PROC FORMAT to label categorical variables (e.g. Dementia and Funding)
- Used PROC MEANS to generate summary statistics
- Performed frequency and cross-tab analysis using PROC FREQ
- Created visualisations including histograms and boxplots

## Tools Used
- SAS (Statistical Analysis System)

## Key Learning
- Data cleaning and preparation techniques
- Basic statistical analysis
- Data visualisation methods
- Understanding relationships between variables

## Output
The analysis provided insights into hospital stay duration and highlighted patterns related to funding type and admission period.
